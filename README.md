# 趣味应用小游戏-在线PK

*******************


## 运行环境
iOS 9.0+

## 功能简介
抢红包，抢茅台等拼的不单单是网络，手速不行也白瞎，该小游戏可以验证自己的手速（每秒点击次数），以及实时PK在线用户的手速，为以后的抢红包做准备

## 资源说明
