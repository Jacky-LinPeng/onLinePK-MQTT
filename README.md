# 趣味应用小游戏-在线PK

*******************


## 运行环境
iOS 9.0+

## 功能简介
抢红包，抢茅台等拼的不单单是网络，手速不行也白瞎，该小游戏可以验证自己的手速（每秒点击次数），以及实时PK在线用户的手速，为以后的抢红包做准备
同步在线用户的每秒点击数，并同步点在状态与点赞爱心动画。

## 资源说明
图标，ICON，封面等资源均为开发使用，并非商业用途。
